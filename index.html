Consensus Algorithms

What is a Consensus Algorithm?

A fundamental protocol in distributed computing that allows a network of independent computers (nodes) to agree on a single, shared state or transaction history, ensuring trust and security without a central authority.

⚙️

How It Works

1

Transaction Broadcast

Users sign and submit transactions, which propagate to all nodes in the network.

2

Block Proposal

A designated or competing validator compiles pending transactions into a new candidate block.

3

Network Validation

Nodes verify the proposed block using cryptographic rules (e.g., PoW math, PoS signatures).

4

Consensus & Commit

Once validation succeeds, nodes append the block to their local copy of the blockchain ledger.
